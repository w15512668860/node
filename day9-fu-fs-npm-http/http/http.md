#### http 超文本传输协议
http:Hyper Text Transfer Protocal
超文本传输协议：用于从web服务器传输超文本到本地浏览器的传送协议。
协议：规则

超文本：html  css  js 图片  音频  视频 ..

传输：从服务器向浏览器发送超文本

简单言之：从服务器向浏览器发送超文本的一种协议

#### http的特点：

1>浏览器可以向服务器发送数据
2>服务器也可以像浏览器发送数据
3>发送请求的主动方永远是浏览器
4>服务器不能主动向浏览器发送请求

#### http请求：

1>ajax

2>地址栏输入地址按回车

3>a标签 href

4>img audio video的src属性

5>link href 

6>script src

7>form表单  action=""
